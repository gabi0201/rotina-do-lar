# 🎀 Rotina Inteligente do Lar

Um sistema visual, no clima Barbiecore, para organizar sua casa com pequenas tarefas diárias — menos de 1 hora por dia, sem sobrecarga.

## ✨ O que o app faz

- **Início** — saudação do dia, progresso das tarefas de hoje e lembrete diário
- **Rotinas** — a semana inteira já planejada, organizada por dia
- **Tarefas** — checklist completo com pontuação, com opção de adicionar ou remover tarefas
- **Ambientes** — tarefas separadas por cômodo (cozinha, sala, quarto, banheiro, área de serviço)
- **Cardápio** — planejamento semanal de café da manhã, almoço, jantar e lanches
- **Lista de compras** — lista organizada por categoria, com itens marcáveis
- **Pontos** — sistema de pontuação e níveis para transformar organização em hábito 💎

## 📱 Como usar

Acesse o link do GitHub Pages pelo navegador do celular e adicione à tela inicial para usar como um app. Todo o progresso (tarefas concluídas, pontos, cardápio e lista de compras) fica salvo automaticamente no navegador.

## 🛠️ Tecnologia

HTML, CSS e JavaScript puros — um único arquivo (`index.html`), sem necessidade de instalação ou servidor.
